# hubot-giphy

A hubot script that searching gif

See [`src/giphy.coffee`](src/giphy.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-giphy --save`

Then add **hubot-giphy** to your `external-scripts.json`:

```json
["hubot-giphy"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
